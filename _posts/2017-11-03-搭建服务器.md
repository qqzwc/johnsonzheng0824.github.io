- ### 一般操作
    - apt-get update 更新源
    - apt-get upgrade 更新已安装的包
- ### 静态网站
    - apt-get install apache2
    - service apache2 restart
    - 在 /etc/apache2/sites-available/000-default.conf 更改默认路径
- ### 安装 redis
    > 安装 gcc
    - yum install gcc
